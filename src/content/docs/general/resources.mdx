Below you'll find the best resources that will help you become a **security researcher**.

## Websites and articles
- [Understanding the vulnerabilities in WordPress](https://patchstack.com/articles/common-plugin-vulnerabilities-how-to-fix-them/) - a trought article by **Dave Jong** describing all the common plugin vulnerabilities with explanations how to fix them.
- [Dockerized WordPress Debug Setup](https://github.com/dimasma0305/dockerized-wordpress-debug-setup) - Setting up the right environment for security by **Dimas Maulana**
- [WordPress plugin security testing cheat sheet](https://github.com/wpscanteam/wpscan/wiki/WordPress-Plugin-Security-Testing-Cheat-Sheet) - a cheat sheet heavily used by the **WPScan Team** when conducting penetration testing against WordPress plugins.
- [WP Directory](https://wpdirectory.net/) - WordPress directory with a possibility to search it using regex. Great when searching for potentially vulnerable code across all plugins.
- [Awesome AppSec](https://github.com/paragonie/awesome-appsec) - a curated list of resources for learning about application security

## Communities
- [Patchstack Community](https://discord.com/servers/patchstack-alliance-1024691600619745334) - Discord-based, with a focus on security (mainly WordPress) and development
